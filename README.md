## Features
- Secure authentication with Supabase
- Create, edit, and delete learning resources
- Organize resources by custom categories
- Search resources by keyword
- Share a resource 
- Clean, responsive UI with Shadcn components

##  Why I Built This
As a developer constantly learning new technologies, I needed a centralized place to organize articles, documentation, and resources I find valuable. StudyLog solves this by providing a simple, fast way to save and retrieve learning materials.

- Database design for relational data
